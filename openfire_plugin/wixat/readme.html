Wixat Plugin Readme

Overview

The presence plugin is a service that provides simple user creation by using crypted information

Installation

Copy wixat.jar into the plugins directory of your Openfire installation. The plugin will then be automatically deployed. To upgrade to a new version, copy the new presence.jar file over the existing file.

Configuration

The wixat plugin can be configured via the Openfire Admin Console. Use the configuration page to specify the secret word (only your applications should know it) and the crypto key. This key must be the same in your client application.

Using the Plugin

Create an user: [hostname]/plugins/wixat/create
Check if exists an user: [hostname]/plugins/wixat/check

The following parameters can be passed into the request:

phoneDescription
phoneRequiredThe crypted phone number (it will be the username). It also must be encoded in base64
secretRequiredSecret word specified in the plugin configuration
passwordRequired when creation Crypted user password. It also must be encoded in base64
